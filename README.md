# Customer Churn Prediction System

A production-ready machine learning system for predicting customer churn with an interactive Streamlit dashboard.

## 🚀 Features

- **Real-time Predictions**: Get instant churn predictions with probability scores
- **Interactive Dashboard**: User-friendly interface with sliders and selectors
- **Feature Engineering**: Automated feature engineering matching training data
- **Business Insights**: Actionable recommendations for customer retention
- **Production Ready**: Deployable on Streamlit Cloud, Hugging Face, or Render

## 📊 Model Performance

- **Accuracy**: 91.5%
- **Precision**: 92.3%
- **Recall**: 91.5%
- **F1 Score**: 90.7%
- **AUC-ROC**: 80.2%

Best Model: **Random Forest**

## 🛠️ Tech Stack

- **Frontend**: Streamlit
- **ML Framework**: Scikit-learn, XGBoost
- **Data Processing**: Pandas, NumPy
- **Deployment**: Streamlit Cloud
